데이터베이스 9장

데이터 검색의 일반적인 형식은 SELECT FROM WHERE/ GROUP BY +HAVING / ORDER BY로 구성이 된다. 전체 테이블에 해당하는 부분은 FROM절에 적어주고 WHERE에 검색할 조건을 적어준다. SELECT 는 테이블에서 조건에 해당하는 값들을 검색하는 기능을한다. GROUP BY는 FROM절을 열로 분할하는 기능을 가지고 있고 Having은 group에 대한 조건을 나타내므로 항상 group과 같이 쓰인다. 마지막으로 order by는 불러올 값을 정렬하고자할때 사용하며 오름차순 내림차순으로 정렬이 가능하다

데이터 검색 중에서 부속질의문은 특수하게 순서를 명시하고 싶을 때 사용하며 질의문에 중첩되어 사용이 된다.부속질의문이 포함되어 있는 질의문을 중첩 질의문이라고 한다. 예를들어 where문 안에 in이 사용될때는 조건을 만족하는 리스트들을 가져오므로 여러개가 나올 수 있다. not in 이 나올때는 조건문을 만족하지 않는 경우를 가져오면 된다.
